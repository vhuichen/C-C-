plugins

plugins

安装插件需要GitBook目录创建 book.json文件

默认插件

{
    "livereload"     热加载插件
    "highlight"     语法高亮插件
    "search"             搜索插件
    "lunr"                 搜索插件后台服务
    "sharing"         分享插件
    "fontsettings"         字体设置插件
    "theme-default"     主题
}

如果要移除默认插件，则插件名称前面加一个-

{
    "-sharing"         移除分享插件
}

安装插件

gitbook install
