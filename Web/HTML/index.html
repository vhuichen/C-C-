HTML

超文本标记语言；HTML并不是一个种编程语言，而是一种标签语言，使用标记标签来描述网页。

<html>标签表示整个HTML文档；

<head>标签包含文档的元数据（如标题和引用的样式表）；

<body>标签包含实际的页面内容。

HTML标签不区分大小写

<!DOCTYPE html> <!-- 描述当前的文件是一个 HTML5 的文件 -->
<html lang="en"> <!-- 语言为en -->
<head>
      <!-- 编码格式 -->
    <meta charset="UTF-8"> 
      <!-- viewport 指设备屏幕上用来显示网页的区域； 设置可视区和设备宽度等宽；设置初始缩放为不缩放 -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>第一个页面</title>
</head>
<body>
      hello world
      1. 输入换行之后不会真的换行，而是产生一个空格<p></p>
2. 不管中间有多少个连续空格最终只会显示一个<p></p>
3. 此时需要段落就要通过段落标签实现<p></p>
  1. 段落标签会换行，也会有一定的间隙<br>
2. 如果不需要间隙，就可以通过换行标签实现<br>
</body>
</html>

表单标签

action：提交表单数据时的目标 URL

method：提交表单数据时使用的 HTTP 方法（通常是 GET 或 POST）。

<form action="https://xxx.com" method="get">
  <!-- for 表示绑定；当用户点击这个标签时会自动将焦点转移到 id="name" 的 input 上 -->
  <label for="name">Name：</label>
  <!-- type：类型；id：唯一标识；name：提交时绑定的名字 required：表示必须要输入 -->
  <input type="text" id="name" name="username" required><br>
