if

if

用法

// 遍历某个目录下所有文件
downdir() {
    for file in `ls $1`
    do
        if [ -d $1/$file ]
        then
            downdir $1/$file
        else
            echo $1/$file
        fi
    done
}

主要参数

-e 如果 file 存在，则真  
-d 如果 file 为目录，则真

-eq 等于
-ne 不等于
-gt 大于
-ge 大于等于
-lt 小于
-le 小于等于
