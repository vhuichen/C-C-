打包失败原因记录

Fastfile 文件为 utf-8 格式的，不小心弄成其他格式后，打包失败

解决方式：先将文件转为 utf-8 格式并保存在别的地方（直接替换 git 会当成同一个文件，无法提交到远程打包机），将旧的文件移除并 git 提交改动，再将 utf-8 格式文件放进来后再次 git 提交改动即可。
