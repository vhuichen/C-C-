RxSwift

RxSwift

take

//1. combineLatest 将3个事件合并成一个事件（3个都需要触发了，最终才会触发）
//2. 闭包，负责组合成新的事件
//3. take 设定条件，满足条件才继续触发事件
//4. flatMap 转换成新的 Observable 序列
self.rxResult = Observable.combineLatest(
    rxEvent1,
    rxEvent2
    rxEvent3,
) { value1, value2, value3 -> (String, String) in
    if let value1 {
        return (value1, value2)
    }
    return (value2, value3)
}.take(while: { advice1, advice2 in
    advice1.count > 0 && advice2.count > 0
}).flatMap { advice1, advice2 -> Observable<Bool> in
    self.commit(advice1, advice2)
}

merge

// 任意一个按钮点击，都会触发事件。注意区别：combineLatest
Observable.merge(aButton.rx.tap.asObservable(), bButton.rx.tap.asObservable()).subscribe(onNext: { _ in
    //...
}).disposed(by: disposeBag)

// 返回的就是其中一个事件的参数，不需要额外处理
Observable.merge(
    rxSelectEvent1.asObservable(),
    rxSelectEvent2.asObservable(),
    rxSelectEvent3.asObservable()
).subscribe(onNext: didSelectModel).disposed(by: disposeBag)

combineLatest

// 必须所有元素都包含一个事件，不然不会触发
// 将多个 Observables 组合起来,统一提供订阅
// 当任意一个源 Observable 发射一个新的元素时，新 Observable 会发射一个包含所有源 Observable 最新元素的元组
Observable.combineLatest(
    rxAdInfo,
    rxTopList,
    rxIsLogin
).subscribe(onNext: { [weak self] _ in
    self?.dosomething()
}).disposed(by: disposeBag)

// 比如下面一直点击其中一个按钮，不会触发事件，只有两个按钮都点击过一次，后面任意点击才会触发事件。
// 如果点击单个需要出发，可以参考 merge
Observable.combineLatest(aButton.rx.tap, bButton.rx.tap).subscribe(onNext: { _ in
    //...
}).disposed(by: disposeBag)

Observable

Observable<T> 这个类就是 Rx 框架的基础，称为可观察序列；需要Observer（观察者）来订阅它，来接收Observable<T>发出的Event

let observer: AnyObserver<String> = AnyObserver { event in
    switch event {
        case .next(let data):
            print(data)
        case .error(let error):
            print(error)
        case .completed:
            print("completed")
    }
}

let observable = Observable.of("A","B","C");
observable.subscribe(observer);

//输出结果
A
B
C
completed

DisposeBag

把 DisposeBag 看做一个容器, 将所有订阅行为存放到容器中, DisposeBag 容器对象在 dealloc 的时候，会对其中存储的所有订阅行为执行 dispose() 方法进行销毁。
