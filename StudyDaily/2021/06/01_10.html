git reset --hard 1e5b9be

1e5b9be指向要退回的下一个节点。此退回会不会保留之前的改动。本质上只是移动了HEAD指针。

//回退到某个提交，不暂存
git reset HEAD@{1}  
git reset --mixed HEAD@{1}
git reset HEAD~1

//回退到某个提交，暂存
git reset --soft HEAD@{1}
git reset --soft HEAD~1

//回退到某个提交，并删除之前的所有更变
git reset --hard HEAD@{1} 
git reset --hard HEAD~1
